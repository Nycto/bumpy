# Bumpy - 2d collision library for nim.

Based on the book http://www.jeffreythompson.org/collision-detection/table_of_contents.php


Supported collisions:

Shape         | Point         | Circle        | Rectangle     | Segment       | Polygon       | Triangle      |
------------- | ------------- | ------------- | ------------- | ------------- | ------------- | ------------- |
Point         | ✅           | ✅           | ✅           | ✅           | ✅           |               |
Circle        | ✅           | ✅           | ✅           | ✅           |               |               |
Rectangle     | ✅           | ✅           | ✅           | ✅           |               |               |
Segment       | ✅           | ✅           | ✅           | ✅           |               |               |
Polygon       | ✅           |               |               |               |               |               |
Triangle      |               |               |               |               |               |               |
